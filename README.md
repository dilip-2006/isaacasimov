# 🧪 IssacAsimov — Lab Component Issue & Management System

IssacAsimov is a **digital platform for managing laboratory components**, replacing the traditional paper-based issue register.  
Students can request components for projects, and lab staff can approve, issue, track returns, and manage inventory — all in one place.

---

## 🔍 Key Features

- Component catalog with availability status  
- Student login & request system  
- Admin approval and issue tracking  
- Return logging & overdue reminders  
- Searchable history of issued items

---

## 🎯 Why This Was Built

Engineering and robotics labs lose time and resources due to **manual registers and unclear tracking**.  
IssacAsimov improves **transparency, accountability, and efficiency** in labs and makerspaces.

---

## 🛠 Tech 

- HTML / CSS / JavaScript  
- Local server backend  
- Database for components, users, and issue logs
